Plateful

Featured
Swift Deepgram A2A Agents LiveKit Python RESTful APIs FastAPI ChromaDB Claude API

Created an iOS app to generate curated meal plans based on calories and macros using Claude. Integrated LiveKit to develop a live voice assistant that provides recipe instructions, sets timers, and offers cooking tips.

Developed an agent that generates grocery lists according to the created recipes, featuring a buy button that utilizes the Kroger API to create a cart. Implemented automatic tracking of perishables, sending notifications when items are close to expiry, and prioritizing them in future recipes.

Built with multiple agents that communicate with each other using the uAgents framework from Fetch.AI, RAG architecture with ChromaDB for personalization, and LlamaIndex for context-aware voice assistance.
